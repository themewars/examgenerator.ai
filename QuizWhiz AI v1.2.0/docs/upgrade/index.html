Upgrade Guide

Before starting upgrade we recommend to back up your all files and export your current database. so in case if anything goes wrong, you can restore the previous version

  1. Backup your current code into new folder.Following things must need to backup :File - .env
  2. Folder - Any new languages that you added in resources/lang/ or lang
  3. Folder - public/uploads/
  4. Backup & Export your database via phpMyAdmin
  5. Delete all your existing code (Before this make sure you performed the step 1)
  6. Upload the latest dist.zip to the server root directory & Extract the zip
  7. Put old .env file back from your backup
  8. Copy lang folder back to the root directory again from your backyup folder
  9. Copy resources/lang folder back from your backyup folder
  10. Copy public/uploads folder back from your backyup folder
  11. Edit .env and change UPGRADE_MODE=true
  12. Run a database upgrade script https://{YOUR_URL}/upgrade/database
  13. Edit .env and change UPGRADE_MODE=false
  14. Verify that latest version is visible in footer

Check everything is working fine.
